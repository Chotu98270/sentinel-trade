# 📊 Sentinel Trade Bot

An automated trading bot built using Python that monitors stock/crypto prices and performs simulated (paper) trades based on RSI strategy.

---

## 🚀 Features

- 📈 RSI-based trading strategy  
- 💰 Paper trading (buy/sell simulation)  
- 📊 Real-time profit tracking  
- 🧾 Trade logging system  
- 📉 Interactive dashboard (Streamlit)  
- 🔄 Multi-stock support  

---

## 🛠️ Tech Stack

- Python  
- yfinance  
- pandas  
- matplotlib  
- streamlit  

---

## ▶️ How to Run

### 1️⃣ Install dependencies
```bash
pip install yfinance pandas matplotlib streamlit
2️⃣ Run trading bot
python main.py
3️⃣ Run dashboard
streamlit run dashboard.py
📸 Output
📜 Trade history table
📈 Profit graph
🔥 Last trade details
💰 Balance & Profit metrics
🧠 Strategy Used
RSI < 40 → BUY
RSI > 60 → SELL
Otherwise → WAIT
📌 Note

This is a paper trading bot (no real money involved).
